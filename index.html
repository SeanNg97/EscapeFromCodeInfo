Escape from Code is a game designed for educational purposes for high school students whose ages range from 13 to 17 years old to learn to code easily and interestingly. The game genre is an educational puzzle game. Programming C will be used for learning material. In the game, the player needs to answer and search for the answer in a room or a space that hiding at the corner of every room based on the question given. Players need to check the question on the board and pick up the box that needs to be inserted as the method to answer the questions. The game was viewed in the first-person view. Level design at three levels with different worlds.
